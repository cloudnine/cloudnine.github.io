Emacs Tutorial, Exporting Org-Mode Files To PDF, Episode 1

Author: Yuri Tricys |   Nov 6, 2020  |    emacs, learning center, video
Emacs Tutorial, Exporting Org-Mode Files To PDF, Episode 1 The above video demonstrates inserting text and images into an org-mode file. I use a function in emacs initialization to pull images from a specified directory into org-mode files. See below. The first function, "get-newest-file-from-dir" is used in the second function, "insert-org-image". It's the second function I'm calling from a hydra when I pull images from the downloads directory. With a little more code you can easily combine downloading and pulling images into org-files, in fact the emacs package easy-hugo has a download, process, and insert image into your blog (markdown or org file) function baked into it.
